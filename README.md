# TRACE Search #
TRACE library for third-party account discovery.

## Building and running on localhost ##

First install dependencies:

```sh
npm install
```

To start:
```sh
npm start
# Open the browser to localhost:8000

# OR

node dist/bundle.js
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running ##

```sh
node dist/bundle.js
```
